# Bludit Theme

Минималистичная тема для Bludit CMS в стиле playtester.io.

## Возможности

- Светлая и тёмная тема (с сохранением в localStorage)
- Фиксированная ширина 980px, центрирование
- Навигация: Explore, Категории, Обо мне, Блог, выпадающее меню More
- Кнопки справа: Поиск, переключатель темы, Добавить (для администратора), Профиль
- Адаптивный дизайн
- Страницы: главная, статья, категория, поиск

## Установка

1. Скопируй папку `bludit-theme` в `/bl-themes/` твоего Bludit
2. Переименуй папку если нужно (например `minimal`)
3. Войди в панель администратора Bludit
4. Перейди: **Settings → Design → Theme** → выбери тему → сохрани

## Структура файлов

```
bludit-theme/
├── index.php              ← главный шаблон (роутинг)
├── metadata.json          ← метаданные темы
├── languages/
│   ├── en.json            ← английские строки
│   └── ru.json            ← русские строки
├── php/
│   ├── home.php           ← список постов
│   ├── page.php           ← одиночная страница/пост
│   ├── category.php       ← страница категории
│   └── search.php         ← результаты поиска
├── css/
│   └── style.css          ← все стили
└── js/
    └── theme.js           ← тема, дропдауны, поиск
```

## Страница "Обо мне"

Создай статическую страницу в Bludit со slug `about` — она автоматически появится в навигации.

## Страницы Privacy / Terms

Создай статические страницы со slug `privacy-policy` и `terms-of-service` — они появятся в выпадающем меню More.

## Кастомизация цветов

Открой `css/style.css` и измени переменные в блоке `:root` (светлая тема) и `[data-theme="dark"]` (тёмная).
