# Свой блоговый движок (PHP)

Минималистичный блог на чистом PHP. Без баз данных — всё в JSON-файлах.

## Возможности
- Статьи с редактором Editor.js (картинки, заголовки, списки, цитаты, код, embeds)
- Категории (CRUD)
- Статические страницы: Обо мне, Правила и т.д. (CRUD)
- Поиск по заголовку, описанию и тексту
- Профиль админа: аватар, имя, bio, смена пароля
- Настройки сайта (название, описание, постов на странице)
- Загрузка картинок локально в `/uploads`
- Защита: CSRF, password_hash, .htaccess для `/data`

## Установка
1. Залейте папку `engine/` на PHP-хостинг (PHP 7.4+)
2. Откройте сайт в браузере — при первом запуске создастся админ `admin / admin`
3. Зайдите в `/?route=admin/login`, **сразу смените пароль** в разделе "Профиль"

## Структура
```
engine/
  index.php          — главный роутер
  config.php         — конфигурация
  .htaccess          — красивые URL + защита
  lib/               — классы (Storage, Auth, Posts, Categories, Pages, Settings, Uploader)
  theme/             — публичные шаблоны и CSS
  admin/             — админ-панель
  data/              — JSON-данные (защищена .htaccess)
  uploads/           — загруженные картинки
```

## Требования
- PHP 7.4+ с расширениями: json, mbstring, fileinfo
- Веб-сервер с поддержкой .htaccess (Apache) — для красивых URL
- Без .htaccess работает на ?route=... параметрах

## Маршруты
- `/` — главная (лента статей)
- `/?route=category/{key}` — категория
- `/?route=post/{slug}` — статья
- `/?route=page/{slug}` — статическая страница
- `/?route=search&q=...` — поиск
- `/?route=admin` — админка
